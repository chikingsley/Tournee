# TBRAC AI Studio App - Example Project

This is an example AI Studio app demonstrating how to integrate Gemini AI with a bowling tournament management system.

**This is part of the Tournee bowling tournament software ecosystem.**

View the original app in AI Studio: <https://ai.studio/apps/drive/11u8Suyg2k3F6AEKobJXSk3MdFTChGMvm>

## About This Example

This example demonstrates:

- AI-powered bowling tournament management
- Gemini API integration
- React + Vite + React Router setup
- Recharts for data visualization
- Lucide icons for UI

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:

   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:

   ```bash
   npm run dev
   ```

## Usage in Tournee

This example shows how AI can be integrated into the bowling tournament workflow for:

- Automated tournament recaps
- Intelligent bracket management
- AI-assisted event planning
- Smart bowler recommendations
